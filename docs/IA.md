# Information Architecture

## Sitemap

```
/                     Landing
/login                Sign in
/register             Create account
/onboarding           Setup wizard
/dashboard            Home hub
/routine              Smart Routine (flagship)
  ├── Weekly view (default tab)
  ├── Daily timeline
  └── Analytics + AI optimizer
/tasks                Task management
/vault                Encrypted secrets
```

## Navigation
Primary sidebar: Dashboard, Routine, Tasks, Vault

## User Roles
- USER: default access to all app features
- ADMIN: reserved for future admin panel
